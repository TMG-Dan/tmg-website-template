# /tmg-developer - Custom Feature Development

Add custom features and functionality beyond standard templates.

## Prerequisites

- Project initialized via `/tmg-setup`
- Basic site structure in place

## Workflow

1. Prompt for:
   - Feature description in natural language
   - Page or component to modify
   - Any third-party integrations needed
2. Analyze request and plan implementation
3. Implement requested functionality
4. Test and validate

## Example Requests

- "Add a booking calendar to the contact page"
- "Create a property search filter"
- "Add Google Maps integration"
- "Build a custom pricing calculator"
- "Add a before/after image slider"

## Common Integrations

- Google Maps
- Calendly / booking systems
- Social media feeds
- Payment processing (Stripe)
- Email marketing (Mailchimp, ConvertKit)
- Live chat widgets

## Output

- Custom feature implemented
- Integration configured
- Component added to relevant page(s)

## Next Steps

Continue with `/tmg-developer` for more features or `/tmg-deploy` to ship
